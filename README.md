# Featured Events ‚Äì Case Study

This project is a **single-page, responsive ‚ÄúFeatured Events‚Äù website** built as part of the **Web Development Internship case study for Dynamics 360**.

The goal of this project is to demonstrate modern web development practices, clean UI design, responsiveness for mobile and tablet, desktop, and component-based structure.

---

## Objective

To build a responsive and interactive webpage that allows users to discover upcoming local events in a clean and user-friendly layout.

---

## Features Implemented

### üîπ Core Requirements
- Responsive **Navigation Bar** with links to:
  - Home
  - Events
  - Contact
  - About
  - Footer
- **Hero Section** with heading:
  > *Discover Events Near You*
- **Featured Events Section** displaying multiple event cards
- Each **Event Card** includes:
  - Event name
  - Image
  - Date & time
  - Location
  - Short description
  - "Read More" button (To see details of the events) 
  - ‚ÄúRegister‚Äù button (UI only)

### Design & Responsiveness
- Clean, modern UI using **custom CSS + Bootstrap + Font Awesome + Google Fonts**
- Fully responsive layout (mobile, tablet, desktop)
- Smooth hover effects and animations
- Glassmorphism-inspired design for cards and sections

### Bonus Features
-  **Search bar** to filter events by name
-  **Dynamic event data** loaded from a dummy JSON file
-  **Event detail page** for better user experience
-  **Google Map** to show event location/ venue
-  **Toastify** Library to show messages/ alerts
- Modular **component-based CSS architecture**
